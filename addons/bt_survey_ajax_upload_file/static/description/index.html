Ajax Survey Upload File

This Module enabling survey upload file using ajax on Survey pages.

Key Highlights

  • Can be setup on survey question as upload file
  • File upload with limit - Maximum upload file size (MB)
  • Allow upload multiple file Yes or No, If Yes allowed for multiple file to upload and No only allowed one file to upload
  • Can setup Mandatory to upload file or optional to upload file

User Guide

  • 1. Add question as upload file.

  • 2. Add Validation for Maximum upload file size (MB) and Allow upload multiple file.

  • 3. Add constraints for Mandatory Answer.

  • 4. Test your survey for upload file.

  • 5. Upload files.

  • 6. Review your answers.

  • 7. See result of Survey.

  • 8. View of uploaded files on user input.
